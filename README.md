## 📖 使用说明

### Windows 用户
1. 📥 下载 `AugmentCleaner-Windows.zip`
2. 📂 解压到任意目录
3. 🖱️ 双击 `AugmentCleaner.exe` 运行
4. 🛡️ 如果被杀毒软件拦截，请添加到白名单

### macOS 用户
1. 📥 下载 `AugmentCleaner-macOS.tar.gz`
2. 📂 解压：`tar -xzf AugmentCleaner-macOS.tar.gz`
3. 🖱️ 双击 `AugmentCleaner` 运行
4. 🔒 如果提示安全警告，请在系统偏好设置中允许

### Linux 用户
1. 📥 下载 `AugmentCleaner-Linux.tar.gz`
2. 📂 解压：`tar -xzf AugmentCleaner-Linux.tar.gz`
3. 🔧 添加执行权限：`chmod +x AugmentCleaner`
4. ▶️ 运行：`./AugmentCleaner`
