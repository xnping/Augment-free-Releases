### 支持的平台
- **🪟 Windows** - `AugmentCleaner-Windows.zip`
- **🐧 Linux** - `AugmentCleaner-Linux.tar.gz`
- **🍎 macOS** - `AugmentCleaner-macOS.tar.gz`

## 🔧 使用说明

1. 下载对应平台的文件
2. 解压到任意目录
3. 运行可执行文件
4. 输入有效的用户名和密码
5. 选择要清理的平台
